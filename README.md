## 💡 什么是 MLTI？

**MLTI**（Machine Learning Thinking Indicator）是一套面向 AI 时代的脑力类型测评系统。

我们相信：在 AI 时代，每个人的思维方式都会被 AI 以不同的方式放大。有人用 AI 做规划，有人用 AI 做创造；有人稳步推进，有人爆发突破。

MLTI 通过 32 道精心设计的选择题，从 4 个维度分析你的脑力类型，生成你的专属 AI 时代脑力画像。

> 🎯 **不是智商测试，不是性格测试，而是你在 AI 时代的竞争力画像。**

## 🧬 四大维度

| 维度 | 对立面 | 你在测什么？ |
|:----:|:------:|:------------|
| 🔍 信息方式 | **F**act ↔ **V**ision | 你从事实出发，还是从趋势出发？ |
| 🧩 思考方式 | **S**tructure ↔ **E**xplore | 你先搭结构，还是先探索可能？ |
| 🤖 AI使用 | **G**uide ↔ **T**ool | 你在引导 AI，还是在使用 AI？ |
| ⚡ 输出方式 | **P**ace ↔ **B**urst | 你是稳定复利型，还是爆发突破型？ |

- 🎯 **32 道精心设计的选择题**，3 分钟完成测评

```bash

git clone https://github.com/Eric-kbo/MLTI.git

npm run dev
```
